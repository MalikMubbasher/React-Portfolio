# Mubbasher Siraj — 3D Portfolio Website

A creative 3D portfolio built with React.js, Three.js, GSAP, and React Native showcasing frontend development expertise.

## 🚀 Tech Stack

- **React.js** + TypeScript + Vite
- **Three.js** / React Three Fiber — 3D character model
- **GSAP** + ScrollSmoother — smooth scroll animations
- **React Three Rapier** — physics-based tech stack section

## 📦 Getting Started

```bash
npm install
npm run dev
```

## 🌐 Deploy to Vercel

1. Push this folder to a GitHub repo
2. Go to [vercel.com](https://vercel.com) → New Project → Import your repo
3. Framework: **Vite**
4. Build command: `npm run build`
5. Output directory: `dist`
6. Click **Deploy**

## 📁 Customization

| File | What to change |
|------|----------------|
| `src/components/Landing.tsx` | Your name and title |
| `src/components/About.tsx` | Bio paragraph |
| `src/components/Career.tsx` | Work experience entries |
| `src/components/Work.tsx` | Project cards |
| `src/components/Contact.tsx` | Email, phone, social links |
| `src/components/Navbar.tsx` | Logo initials, email |
| `src/components/SocialIcons.tsx` | GitHub/LinkedIn URLs |
| `public/Mubbasher_Siraj_CV.pdf` | Your resume PDF |

## 🖼️ Project Images

Replace `public/images/placeholder.webp` with your actual project screenshots.
Name them `project1.webp`, `project2.webp`, etc. and update `Work.tsx` accordingly.
